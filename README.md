# 🏝️ Treasure Island Game (Python)

A fun beginner Python project that creates a simple text-based adventure game. The player must make the right choices to survive and find the hidden treasure.

---

## 📌 Features

* 🧭 **Interactive Story Game**
  Players choose directions and actions that affect the outcome.

* ❓ **Multiple Choices**
  Different paths lead to winning or losing (Game Over).

* 🏆 **Winning Scenario**
  Find the correct path to discover the treasure.

---

## 🧠 Concepts Used

* User Input (`input()`)
* Conditional Statements (`if`, `elif`, `else`)
* String methods (`.lower()`)
* Nested conditions
* Basic game logic

---

## 🚀 How to Run

1. Install Python
2. Save the code in a file (e.g., `main.py`)
3. Run the game:

```id="runisland"
python main.py
```
